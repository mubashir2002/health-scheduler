# Health Scheduler

Health Scheduler is a Flutter-based mobile app for browsing doctors, exploring profiles, booking appointments, and tracking scheduled visits in a clean, animated interface.

## Purpose

The project is designed to make medical appointment booking feel simple and modern. It guides users from onboarding to doctor discovery, appointment booking, and confirmation without a heavy or cluttered experience.

## Tech Stack

- Flutter
- Dart
- Material UI
- Cupertino Icons
- Flutter animations and custom themes

## Core Features

- Animated splash screen and onboarding flow
- Doctor browsing and profile details
- Appointment booking flow
- Booking confirmation screen
- Bottom navigation for main app sections
- Profile and appointments screens
- Responsive layout for mobile devices

## Project Structure

- `lib/main.dart` - app entry point
- `lib/screens/` - all UI screens
- `lib/theme/` - app colors and theme setup
- `lib/models/` - data models
- `assets/images/` - app images and illustrations

## Getting Started

### Prerequisites

- Flutter SDK 3.11 or later
- Android Studio or VS Code with Flutter support
- An Android emulator, iOS simulator, or physical device

### Run Locally

```bash
flutter pub get
flutter run
```

### Run Tests

```bash
flutter test
```

## Notes

- The package name is `health_scheduler`.
- The app title shown in the UI is Health Scheduler.
- This project is currently set up as a private Flutter app and is not published to pub.dev.
